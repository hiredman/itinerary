# lein-itinerary

A Leiningen plugin to run tests using itinerary

## Usage

Put `[com.thelastcitadel/lein-itinerary "0.1.0-SNAPSHOT"]` into the
`:plugins` vector of your project.clj.

    $ lein itinerary
